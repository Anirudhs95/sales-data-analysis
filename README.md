# Sales Data Analysis

## Project Overview
This project analyzes sales data using Python to identify
sales trends and best-performing products.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Total and average sales calculation
- Product and category-based analysis
- Data visualization using charts

## Key Insights
- Electronics category generated the highest revenue
- Laptops were the top-selling product

## Dataset
Sample dataset created for analysis purposes.
